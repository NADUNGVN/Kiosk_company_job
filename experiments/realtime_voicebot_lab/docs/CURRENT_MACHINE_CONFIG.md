# Cấu Hình Máy Hiện Tại

Thông tin này được ghi ngắn gọn để làm bối cảnh cho nhánh thử nghiệm trên máy
Ryzen.

- Hãng/model: Lenovo `83LT`
- CPU: AMD Ryzen 9 8945HX with Radeon Graphics
- Số nhân/luồng: 16 cores / 32 logical processors
- GPU: NVIDIA GeForce RTX 5060 Laptop GPU
- VRAM báo bởi Windows: khoảng 4 GB
- RAM hệ thống: khoảng 16 GB
- Python đang dùng: Python 3.10.0
- Python executable: `.venv\Scripts\python.exe`
- Hệ điều hành: Windows build 26200

Ghi chú: Python `platform` có thể báo chuỗi `Windows-10` do lớp tương thích của
Windows, dù giao diện hệ điều hành thực tế là Windows 11.
